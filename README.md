## Как это работает?

    Схема: Программа определяет структуру данных (заголовок, текст, теги, дата).

    Индексация: Текст разбивается на токены и сохраняется в инвертированный индекс в папке indexdir.

    Поиск: При запросе движок использует алгоритм BM25F, чтобы найти наиболее подходящие документы и подсветить совпадения.

## Требования для запуска

    Python: версии 3.7 или выше.

    Библиотеки: whoosh.

## Установка и запуск

    Клонируйте репозиторий (или просто скопируйте файлы проекта в одну папку).

    Установите зависимости:
    pip install whoosh

    Запустите:
    python main.py

## Структура проекта

    config.py — настройки схемы данных и пути к хранилищу.

    engine.py — ядро системы (добавление документов, логика поиска и подсветки).

    main.py — скрипт с примерами данных и тестовыми запросами.

    indexdir/ — (создается автоматически) папка с бинарными файлами индекса.

## Основные возможности

    Full-text search: поиск по всему объему текста.

    Fuzzy matching: если вы введете "Pyton", система предложит исправление на "Python".

    Highlighter: найденные слова в консоли будут выделены контекстом.

    Smart Update: система не дублирует файлы, а обновляет их в индексе по уникальному пути.
